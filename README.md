## Storybook UI Example

This project is an example how to use storybook to create a company UI library.

#### Tech Stack:
 
 - React 18.2.0
 - Typescript 4.9.5
 - Storybook 6.5.16
 - Styled-Components 5.3.6
 - Webpack 5.75

#### Setup

Start Storybook in port 8080
~~~
$ yarn install
$ yarn start
~~~

Run tests
~~~
$ yarn test
~~~
